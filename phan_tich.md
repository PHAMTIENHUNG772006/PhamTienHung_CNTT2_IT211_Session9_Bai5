Phần 1 : phaan tích


Dễ dàng phân tích & truy vấn: JSON log có cấu trúc key–value rõ ràng, giúp các hệ thống như ELK/Datadog parse tự động, lọc theo field (userId, errorCode…) nhanh chóng, thay vì phải regex text log.

Tích hợp tốt với monitoring: JSON log chuẩn hóa dữ liệu, dễ gửi qua pipeline, dễ thống kê, vẽ dashboard, cảnh báo chính xác hơn so với text log vốn chỉ là chuỗi khó tách.
